By default, this extension will search for magnet links in the page source by looking at any HTML elements that have either:

  • an href attribute containing a magnet link
  • a code element with text that matches an element link

This section allows the addition of more element types containing magnet links in the thier text, or look for a magnet link in elements attributes other than href.

Care should be taken with these options as it's easy to cover a page in links if selectors are too vague!
